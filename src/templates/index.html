Welcome

Boycottree is a website that tracks products subject to boycotts by documenting the links between products, corporations, countries and more. The site is intended to be user editable and crowd-sourced. The software is intended to be open source to allow for feature contribution.

Using the site

Using the site search you will be able lookup a company or product and explore their links across the database. You can also search our catalogue of boycotts. You will be able to search by company name, product name, boycott name or product barcode.

How to help

The site will go through a number of stages, with different types of contributions required at different points. Initially (and for the life of the site) we need coders and designers

If you would like to you can take a look at the source code on GitHub
